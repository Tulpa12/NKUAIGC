## 一、引言

生成式人工智能（AIGC）正在深刻改变内容生产的方式。从文本生成到图像修复，从数字人到视频创作，AI工具让个人创作者拥有了前所未有的表达手段。作为南开大学的一名学生，我希望将AIGC技术与南开大学的百年校史相结合，用技术的力量让尘封的历史影像重获新生。

## 四、创作思路与步骤拆解

### 4.1 创作思路

本作品的创作核心思路为"以AIGC技术为桥梁，连接历史与当下"。具体逻辑如下：

1. **素材层**：从南开大学校史档案与公开资料中选取8张跨越不同历史时期的老照片
2. **技术层**：使用AI图像修复模型对照片进行画质增强，去除划痕、噪点，恢复模糊细节
3. **叙事层**：以B站科普视频的形式，结合数字人播报和修复过程展示，讲述南开百年故事
4. **交互层**：搭建互动网页，让观众通过拖拽滑块亲手体验修复效果，增强参与感和传播性

### 4.2 步骤一：素材收集与筛选

从南开大学官方校史资料、澎湃新闻《一眼百年》专题、南开大学新闻网「图说南开」系列中，收集了150余张南开大学历史影像。经过筛选，最终选定8张照片，涵盖年代如下：

| 序号 | 年代 | 内容 | 筛选理由 |
|------|------|------|----------|
| 1 | 1919 | 南开大学开学纪念合影（周恩来后排左一） | 历史起点，人物重要 |
| 2 | 1920s | 思源堂 | 南开最古老建筑，古典建筑美感 |
| 3 | 1928 | 木斋图书馆 | 文化象征，后续被炸形成叙事呼应 |
| 4 | 1937 | 被日军轰炸后的南开校园 | 视觉冲击力强，家国情怀 |
| 5 | 1946 | 南开返津复校 | 涅槃重生主题 |
| 6 | 1950s | 毛泽东主席视察南开大学 | 标志性历史瞬间 |
| 7 | 1980s | 南开师生校园生活 | 拉近年代距离 |
| 8 | 1990s | 南开大学老校门 | 百年守望，视频结尾定帧画面 |

筛选标准：画面主体清晰（具有辨识度）、照片有不同程度破损（修复前后反差明显）、年代跨度大（覆盖1919至1990年代）。

### 4.3 步骤二：AI图像修复增强

使用restorePhotos.io（基于GFPGAN模型）对8张照片逐一进行修复处理。GFPGAN（Generative Facial Prior GAN）是腾讯ARC Lab提出的盲人脸修复模型，能够有效处理老照片中的划痕、噪点、模糊等问题。

操作过程：
1. 将原照片上传至restorePhotos.io
2. AI自动检测并修复画质缺陷，处理时间约10-20秒/张
3. 下载修复后版本，对比检查效果
4. 对修复力度不足的照片适当重复处理

修复前后效果对比：经过AI修复，多张照片的细节得到了显著提升——人脸轮廓从模糊变为清晰，建筑纹理从一片糊状恢复为可辨的砖瓦结构，历史影像的信息量大幅增加。

### 4.4 步骤三：AI脚本撰写与优化

使用ChatGPT（GPT-4o）进行视频脚本的创作。采用三轮迭代优化策略：

**第一轮**：给定主题和基本要求，生成初始脚本框架
**第二轮**：强化开场视觉冲击、加入真实"翻车"案例（AI上色的文化与政治边界问题）、自然融入招生宣传
**第三轮**：精修语言（口语化、25字/句以内）、标注停顿和情绪、控制总时长7分钟

同时生成AI数字人专用播报稿（精简版，约5分30秒）。

### 4.5 步骤四：AI上色的探索与放弃

在完成画质修复后，曾尝试使用Colorize.cc和palette.fm对照片进行AI上色。但在实际测试中发现了两个严重问题：

1. **国旗识别错误**：Colorize.cc将照片中模糊的五星红旗识别为美国国旗
2. **人像种族偏差**：AI将亚洲面孔修复上色后呈现为西方人特征

这些问题的根因在于——主流AI上色模型的训练数据以西方影像为主，缺乏中国历史场景和亚洲人像的充分训练。出于内容安全和文化尊重的考量，最终决定放弃AI上色环节，专注于画质修复。

这一决策过程本身，成为了视频叙事的重要组成部分——"知道AI的边界，比盲目炫技更重要。"

### 4.6 步骤五：互动对比网页制作

使用Claude 4辅助生成纯HTML/CSS/JS单文件网页，实现"修复前后对比"的拖拽滑块交互效果。

技术实现：
- **布局**：CSS Grid + Flexbox响应式设计
- **交互**：原生JavaScript实现鼠标拖拽和触摸滑动
- **视觉风格**：南开校色（紫金配色）+ 渐变背景 + 淡入动画
- **兼容性**：支持PC端和移动端浏览器

### 4.7 步骤六：视频制作与合成

使用剪映桌面版进行视频后期制作：
1. 导入修复前后的对比素材、修复过程录屏、翻车案例截图
2. 使用剪映数字人功能生成AI播报视频
3. 添加转场效果（硬切 + 淡入淡出）、字幕
4. 选用温暖钢琴配乐营造情感氛围
5. 导出参数：1080p / 30fps / H.264

## 六、关键代码展示

互动对比网页核心交互逻辑（JavaScript）：

```javascript
// 拖拽滑块对比修复前后照片的核心逻辑
(function(){
  document.querySelectorAll('.comparison-wrapper').forEach(function(wrapper){
    var after  = wrapper.querySelector('.img-after');
    var before = wrapper.querySelector('.img-before');
    var line   = wrapper.querySelector('.slider-line');
    var handle = wrapper.querySelector('.slider-handle');
    var active = false;

    function setPos(x){
      var rect = wrapper.getBoundingClientRect();
      var pct = Math.max(0, Math.min(1, (x - rect.left) / rect.width));
      // 右侧显示修复后的照片（clip-path控制可见区域）
      after.style.clipPath  = 'inset(0 ' + (100-pct*100) + '% 0 0)';
      // 左侧显示原始照片
      before.style.clipPath = 'inset(0 0 0 ' + (pct*100) + '%)';
      line.style.left   = (pct * 100) + '%';
      handle.style.left = (pct * 100) + '%';
    }

    // 鼠标事件绑定
    wrapper.addEventListener('mousedown', function(e){
      active = true; setPos(e.clientX); e.preventDefault();
    });
    window.addEventListener('mousemove', function(e){
      if(active) setPos(e.clientX);
    });
    window.addEventListener('mouseup', function(){ active = false; });

    // 触摸事件绑定（移动端支持）
    wrapper.addEventListener('touchstart', function(e){
      active = true; setPos(e.touches[0].clientX);
    }, {passive: false});
    window.addEventListener('touchmove', function(e){
      if(active) setPos(e.touches[0].clientX);
    }, {passive: false});
    window.addEventListener('touchend', function(){ active = false; });
  });
})();
```

完整网页代码见附件 `index.html`。

## 七、创作过程中的挑战与解决方案

### 挑战一：AI上色的文化与政治边界

**问题描述**：使用Colorize.cc等海外AI工具对历史照片进行上色时，出现将五星红旗识别为美国国旗的严重错误。同时，palette.fm等工具在处理亚洲面孔时存在种族偏差，呈现西方人特征。

**解决方案**：果断放弃AI上色环节，将作品聚焦于AI画质修复增强。这一决策不仅规避了内容安全风险，更成为了视频的核心叙事线索——"知道技术的边界，比炫技更重要。AI可以修复像素，但理解不了文化的重量。"

**启示**：国产AI工具在中文语境和中国文化理解上具有不可替代的优势。对于涉及中国历史文化的内容创作，应优先考虑国产AI或对训练数据有明确的本地化把控。

### 挑战二：历史照片质量参差不齐

**问题描述**：8张老照片的破损程度差异较大。部分照片仅轻微模糊，修复效果立竿见影；部分照片存在大面积划痕和缺失，AI修复后出现"过度平滑"现象。

**解决方案**：采用分档处理策略。轻度破损一次修复即可，重度破损适当降低修复强度参数，保留一定的"历史质感"而非追求完全平滑。修复前后对比的视觉冲击力本身，比"修得完美"更重要。

### 挑战三：视频节奏把控

**问题描述**：脚本时长初版超过9分钟，且开场铺垫过长，不符合B站用户前5秒决定完播的行为习惯。

**解决方案**：将开场改为第1秒即出现修复前后对比的无声硬切，让视觉冲击成为第一印象。压缩技术讲解部分，控制总时长在7分45秒。多次试看后微调节奏，确保每个段落都有信息增量或情感推进。

### 挑战四：数字人表达的自然度

**问题描述**：AI数字人在播报长句时语调单一，缺乏自然的情感起伏，影响观众的沉浸感。

**解决方案**：将脚本改写为短句（每句不超过25字），明确标注停顿位置和情绪指示。同时，在剪映中适当调整数字人的语速参数（降速5%），使整体节奏更加从容自然。

## 八、个人感悟与收获

### 技术层面

通过这次创作，我实践了AIGC工具链的串联使用——从图像修复到脚本生成，从数字人制作到网页开发，每一步都有AI参与辅助。最大的体会是：AI不是替代创作者的"自动魔法"，而是放大创作者能力的"超级工具"。真正决定作品质量的，是创作者对工具的驾驭能力和审美判断力。

### 认知层面

Colorize.cc把国旗识别错误的那一刻，对我来说是一次深刻的"技术觉醒"。AI模型的偏见不是技术故障，而是训练数据的文化盲区。作为中国的大学生，在使用全球化的AI工具时，保持文化自觉和政治敏感不是保守，而是必要的能力。这也是南开校训"允公允能"的题中之义——能力越大，判断力的责任越大。

### 传播层面

好内容需要好叙事。AI修复老照片本身是一项技术操作，但要让它产生传播力，需要赋予它故事、情感和价值主张。通过"百年南开"的时间线叙事、修复前后的视觉对比、以及"技术边界"的反思，让一个技术演示变成了一段有温度的文化表达。

### 对南开的情感

修复这些老照片的过程，让我对南开大学有了更深的理解。从1919年开学典礼上的周恩来，到1937年被日寇轰炸的校园，再到1946年的涅槃重生——每一张照片背后，都是一代代南开人的故事。"允公允能、日新月异"这八个字，在这些黑白影像中有了具体的分量。
