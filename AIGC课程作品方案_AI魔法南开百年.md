# 《AI魔法·南开百年》AIGC课程作品完整方案

---

## 一、作品概述

| 项目 | 内容 |
|------|------|
| 作品名称 | 《AI魔法·南开百年》 |
| 作品类型 | AI修复视频 + AI数字人播报 + 互动对比网页 |
| 核心AI技术 | AI图像修复、AI黑白上色、AI数字人生成、AI脚本撰写 |
| 创作主旨 | 以AI技术为桥梁，连接南开的过去与未来，向高考生展示一所有底蕴、有温度的百年名校 |
| 目标受众 | B站年轻用户（16-22岁为主），高考考生及家长，南开校友 |

## 六、核心提示词（含迭代优化过程）

### 6.1 ChatGPT视频脚本撰写

**第一轮（初始版本）**：
```
你是一位资深纪录片导演和B站内容创作者。我需要为一个5-8分钟的
B站视频写播报脚本。主题是"用AI修复南开大学历史老照片并制作
数字人播报视频"。请帮我写一个完整的脚本，包含：
1. 开场钩子
2. 技术过程介绍
3. 修复前后对比展示
4. 个人感悟

要求：符合B站年轻观众的审美，有情感张力，不要太说教。
```

**第二轮（优化迭代）**：
```
第一版脚本过于平淡，需要强化以下几点：
1. 开场必须在前10秒展示修复对比的视觉冲击
2. 中间加入一个"翻车"段落（AI修复失败案例），增加真实感
3. 结尾自然融入高考招生宣传，不要太生硬——用"传承与创新"
   的主题来串联
4. 语言更口语化，减少书面语，增加"好家伙""直接"等B站常用表达
5. 总时长控制在7分钟左右，每部分标注预估时长

请重新生成。
```

**第三轮（精修）**：
```
第二版好多了。再做以下调整：
1. 技术部分增加具体AI工具名称，让技术观众觉得有干货
2. "翻车"部分加入自嘲语气，别太严肃
3. 结尾不要出现"招生"二字，用"我们在南开等你"自然替代
4. 每段标注画面建议和BGM建议
5. 总字数控制在1800字以内（对应7分钟语速）

请按视频脚本格式输出最终版。
```
